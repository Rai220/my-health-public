## Drugs / инструкции

В этой директории собраны файлы по препаратам — справочные карточки с выжимками из официальных инструкций.

### Препараты

- `atomoxetine-strattera.md` — атомоксетин (Strattera) — NRI для СДВГ
- `buspirone-buspar.md` — буспирон (BuSpar) — 5-HT1A агонист для тревожности
- `escitalopram-lexapro.md` — эсциталопрам (Lexapro) — SSRI
- `fluoxetine-prozac.md` — флуоксетин (Prozac) — SSRI
- `guanfacine-intuniv.md` — гуанфацин (Intuniv, пролонг.) — α2A-агонист для СДВГ
- `lamotrigine-lamictal.md` — ламотриджин (Lamictal) — стабилизатор настроения
- `lurasidone-latuda.md` — луразидон (Latuda) — атипичный антипсихотик
- `pregabalin-lyrica.md` — прегабалин (Lyrica) — габапентиноид, показан при ГТР (ЕС)
- `sertraline-zoloft.md` — сертралин (Zoloft) — SSRI
- `trazodone-desyrel.md` — тразодон (Desyrel) — антидепрессант/снотворное
