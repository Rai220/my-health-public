# genetic/ — генетические данные

Сюда кладутся файлы расшифровок геномов (результаты генотипирования и секвенирования).

## Поддерживаемые форматы

- **23andMe** — txt-файл с raw data
- **AncestryDNA** — txt-файл с raw data
- **Whole Genome Sequencing (WGS)** — VCF-файлы
- **Whole Exome Sequencing (WES)** — VCF-файлы
- Другие форматы от лабораторий генотипирования

## Как использовать

1. Скачайте raw data из сервиса генотипирования
2. Положите файл(ы) в эту папку
3. AI-ассистент сможет анализировать генетические данные с помощью инструментов из `tools/`
