# Гипотезы

Гипотезы — это **не факты**, а предположения, которые нужно подтвердить или опровергнуть.

Статусы: `open` / `confirmed` / `refuted` / `partially_confirmed`

## Формат записи

```
### [H-NNN] Краткое название гипотезы

- **Статус**: open | confirmed | refuted | partially_confirmed
- **Дата**: YYYY-MM-DD (когда сформулирована)
- **Область**: фармакология / диагностика / генетика / образ жизни / ...
- **Суть**: Что именно предполагается (1-3 предложения)
- **Основания**: Почему возникла гипотеза (наблюдения, данные, тайминг)
- **Как проверить**: Конкретные шаги для подтверждения/опровержения
- **Связи**: ссылки на MEDS.md, PROBLEMS.md и т.д. (если есть)
- **Обновления**:
  - YYYY-MM-DD — что изменилось
```
